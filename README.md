# Piao

💌一个单词记忆app

缥，帛青白色也。——《说文》

本项目使用uni-app框架基于Vue.js开发，UI采用TM-vuetify，数据库采用uniCloud链接阿里云空间，Dcloud（北京）云打包服务构建
